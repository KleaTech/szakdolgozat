A program a következő paranccsal fordítható:
The program can be complied with this command:

mvn install

A fordításhoz legalább 3.3.9-es verziójú Maven ajánlott, és legalább Java 10 JDK szükséges.
To compile the program at least Maven version 3.3.9 is recommended and at least Java 10 JDK is mandatory.

A futtatás a következő parancs kiadásával történhet:
The program can be run with this command:

mvn spring-boot:run -pl hu.kleatech.jigsaw

Vagy a generált, "exec" végű, nagy méretű .jar fájl futtatásával is indítható az alkalmazás.
Alternatively the program can be started by running the generated, large .jar file ending with "exec".
